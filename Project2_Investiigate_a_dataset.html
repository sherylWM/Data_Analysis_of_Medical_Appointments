Project: Exploratory Data Analysis of No Show Appointment Dataset¶

The dataset used for this project is the No-show appointments dataset. The csv file 'noshowappointments-kagglev2-may-2016.csv' is stored in the same location as the 'Investigate_a_dataset.ipynb' file.

Research Questions¶

Following research questions were selected for this project.

  1. What factors are important for us to know if a patient will show up for their scheduled appointment?
  2. Do patients show up more if appointments are scheduled on weekends or weekdays?
  3. Does waiting time i.e. ScheduledDay - AppointmentDay lead to no shows?

#Converting ScheduledDay column to DateTime format
noshowappointments_df['scheduled_day'] = pd.to_datetime(noshowappointments_df['ScheduledDay'], format='%Y-%m-%dT%H:%M:%SZ')
#Converting AppointmentDay to DateTime format

noshowappointments_df['waiting_time'] = round((noshowappointments_df['scheduled_day'] - noshowappointments_df['appointment_day'])/np.timedelta64(1,'D'),0)
noshowappointments_df['waiting_time'] = abs(noshowappointments_df['waiting_time'].astype('int'))
